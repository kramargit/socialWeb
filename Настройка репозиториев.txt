
Как подготовить, настроить удаленный репозиторий на GitHub:
Чтобы создать новый репозиторий на GitHub, вам необходимо выполнить следующие действия:

Войдите в свою учетную запись GitHub и нажмите значок «+» в правом верхнем углу страницы. В раскрывающемся списке выберите «Новый репозиторий».
На странице «Создать новый репозиторий» вам нужно будет выбрать имя для вашего репозитория и добавить описание (если хотите).
Затем выберите, будет ли ваш репозиторий общедоступным (видимым для всех в Интернете) или частным (видимым только вам и людям, которым вы предоставили доступ). 
Обратите внимание, что для частных репозиториев требуется платная подписка на GitHub.
Вы также можете добавить файл README в свой репозиторий, что является хорошей практикой для объяснения того, о чем ваш проект и как его использовать.
Наконец, нажмите кнопку «Создать репозиторий», чтобы завершить настройку нового удаленного репозитория на GitHub.





Как оформить мой код в этот репозиторий:
После того, как вы создали удаленный репозиторий на GitHub, вам нужно будет извлечь свой код из локального репозитория и отправить его в удаленный репозиторий. 
Вот как это сделать:

Во-первых, вам нужно инициализировать репозиторий Git в вашем локальном каталоге проекта, если вы еще этого не сделали. Для этого запустите команду «git init» в 
каталоге вашего проекта.
Затем вам нужно добавить файлы из каталога вашего проекта в репозиторий Git. Запустите команду «git add». чтобы добавить все файлы в вашем каталоге в репозиторий.
Теперь зафиксируйте изменения, которые вы внесли в свои файлы, с помощью команды «git commit -m 'initial commit'».
Далее вам нужно настроить удаленный репозиторий как удаленный репозиторий Git. Для этого выполните команду «git remote add origin <URL-адрес удаленного репозитория>». 
Замените <URL-адрес удаленного репозитория> URL-адресом вашего удаленного репозитория на GitHub.
Наконец, отправьте свой код в удаленный репозиторий с помощью команды «git push -u origin master». Это отправит ваш код в основную ветку удаленного репозитория.
